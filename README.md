# JK Correctron APFC Smart Platform

A web-based platform designed for learning, calculation, installation, and troubleshooting of Automatic Power Factor Correction (APFC) systems, specifically for Accuvar Trinity KVAR-based controllers.

---

## Overview

This platform combines technical training, simulation, and practical tools into a single interface. It is built to support engineers, technicians, and students working with APFC panels and power factor management systems.

---

## Key Features

- Learning module covering power factor fundamentals, installation, and troubleshooting
- kVAR calculation tool for capacitor bank sizing
- Step-by-step troubleshooting assistant
- Guided installation wizard with wiring references
- Simulation dashboard for PF, kW, kVAR, and system behavior
- Searchable digital manual for Accuvar Trinity controller

---

## Technology Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Chart.js
